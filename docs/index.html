Frequently Asked Questions

How do I start a new game?

Tap the + button on the main screen. Enter player names, configure any optional rules, then tap Start Game.

Can I edit a score after it's been entered?

Yes. Open the game and tap any completed round to reopen the round editor and correct bids or tricks.

What is the Dealer Forbidden Bid rule?

When enabled, the dealer cannot bid the value that would make total bids equal the number of cards dealt. This guarantees at least one player will miss their bid each round.

What does Reserve Trump Card do?

When enabled, one card is set aside face-up as the trump suit indicator. This reduces the maximum hand size by 1 card.

How is scoring calculated?

Bid 0 and take 0 tricks → +10 pts. Bid n and make it exactly → +50 + (10 × n) pts. Miss by d tricks → −10 × d pts.

How do I delete a game?

On the main game list, swipe left on a game and tap Delete.

Does the app store any data online?

No. All data is stored on your device only. The app makes no network requests and collects no personal information.
